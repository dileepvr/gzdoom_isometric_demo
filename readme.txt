MyStandaloneGame - GZDoom Isometric Demo

A demo for isometric mode in GZDoom: https://github.com/dileepvr/gzdoom_isometric/tree/isocam_with_actors
Expected to become a feature starting version 4.12 onwards.

Just run `make` to generate an ipk3 file in the build directory.

Game template for GZDoom-based games created by Dileep V. Reddy
Based off of a Game template created by Nash Muhandes
